Sistemas de Información Geográfica¶

Este es un libro libre sobre Sistemas de Información Geográfica.

Además de consultarlo online, puedes comprar ejemplares impresos o descargarlo en formato pdf.

También puedes consultar el código fuente del libro (en formatos LaTeX y Sphinx).

Para cualquier comentario, puedes contactar con el autor.
